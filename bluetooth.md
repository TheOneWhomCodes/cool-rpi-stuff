

# Bluetooth stuff!!


## Basics

to enter the bluetooth terminal type:

´´´
bluetoothctl
´´´
ctl stands for control, cool huh

to enable bluetooth, type:

```
power on
```

and to turn it off

```
power off
```

If you want to scan for nearby devices that're using bluetooth, you can use this

```
scan on
```
